== Description ==

### PopAd

The Ultimate PopUp Ads Plugin! With a Lot of Features.. Discover it by yourself.

### The Features

1. Create PopUp Ads easily.
2. Unlimited PopUp Ads.
3. PopUp Ads different for every post or page or any custom post type.
4. General PopUp Ad (display popup ad on the entire website, all pages, all posts, homepage, etc).
5. Media PopUp: YouTube PopUp, Vimeo PopUp, SoundCloud PopUp.
6. Autoplay option for Media PopUp (youTube, vimeo, soundcloud).
7. Responsive Design.
8. Compatible with [Preloader](https://wordpress.org/plugins/the-preloader/).
9. AdSense Extension.
10. Anti AdBlock Extension.
11. Statistics of Impressions (unique).
12. Custom time for every PopUp Ad.
13. Exclusion Options (for example exclude homepage, exclude editor user, etc).
14. Easy to use.
15. And more features!.

== Installation ==

### Installation

1. Upload 'popad' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Read [Explanation of Use](http://wp-plugins.in/PopAd_Plugin)
